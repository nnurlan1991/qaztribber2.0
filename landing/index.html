QazTriber — десктопное приложение для транскрибации казахской, русской и смешанной речи. Аудио обрабатывается локально на вашем устройстве. Никаких серверов, никакой слежки.

macOS: если приложение «повреждено»

Apple блокирует приложения без сертификата разработчика. Это не реальное повреждение. Откройте Terminal и выполните:

xattr -cr /Applications/QazTriber.app

После этого приложение запустится. Это разовая операция.

Полностью офлайн

ИИ модель для распознавания работает локально. Аудио не покидает ваше устройство. Интернет нужен только для первой загрузки моделей.

Три языка

Казахский, русский и смешанная речь (KZ+RU). Выбирайте язык записи для максимальной точности распознавания.

Запись и импорт

Записывайте с микрофона или импортируйте WAV, MP3, M4A, FLAC. Редактируйте, копируйте и экспортируйте результат в TXT.

Apple Silicon ускорение

На Mac с M1/M2/M3/M4 используется Metal Performance Shaders — расшифровка в 3 раза быстрее. На Windows — CPU оптимизация.

Две модели

Быстрая (220M) для черновиков и точная (600M) для максимального качества. Переключайтесь в один клик.

Приватность

Никаких аккаунтов, никаких облаков, никакой телеметрии. Ваши записи остаются только у вас.
